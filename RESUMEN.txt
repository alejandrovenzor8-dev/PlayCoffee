🚀 CÓMO COMENZAR (PRIMERA VEZ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Paso 1: Instalar PostgreSQL 16
  → Ver: POSTGRESQL-SETUP.md
  → Descargar: https://www.postgresql.org/download/windows/

Paso 2: Ejecutar setup
  → powershell: .\setup.ps1

Paso 3: Crear base de datos
  → psql -U postgres -c "CREATE DATABASE playcoffee;"

Paso 4: Ejecutar migraciones
  → pnpm db:migrate

Paso 5: Iniciar proyecto
  → pnpm dev

  • "Port already in use" → netstat -ano | findstr :3000
  • "Can't reach database" → Get-Service postgresql*
  • "Prisma Client not generated" → pnpm db:generate
